MultiHashMappingArray

class MultiHashMappingArray(context: QueryContext, bindings: Set<String>) : MappingArray

An array useful for storing a series of mappings, capable of joining with other mappings using the hash join algorithm. Hash tables are created for every binding name passed in the constructor.
